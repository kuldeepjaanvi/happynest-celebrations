# Assets Folder

This folder contains all product images for the HappyNest Celebrations website.

## Required Images

Based on `data/catalogue.json`, you need the following images:

## Image Guidelines

- **Format**: JPG or PNG
- **Recommended Size**: 800x600px (or similar 4:3 aspect ratio)
- **File Size**: Keep under 500KB per image for fast loading
- **Naming**: Use lowercase with underscores (e.g., `plum_cake.jpg`)

## Adding New Images

1. Add your image files to this `assets/` folder
2. Update `data/catalogue.json` with the correct image path
3. The path should start with `assets/` (e.g., `assets/my_image.jpg`)

## Placeholder Images

If an image is missing, the website will display a placeholder. Make sure to add all product images for the best user experience.
